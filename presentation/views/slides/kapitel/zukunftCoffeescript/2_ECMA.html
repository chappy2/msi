ECMAScript 6. Edition

Features die aus CoffeeScript bekannt sind:

  • Blocked-Scoped Variablen
  • Arrow-Funktionen
  • Standardwerte für Parameter
  • Template Strings
  • Klassen

Allein die Syntax unterscheidet sich zu CoffeeScript

  • 7. Edition von ECMAScript ist zwar die aktuelle, aber die 6. Edition hat den JavaScript Standard grundlegend verändert.
  • Ziel von ES6 war es diverse Praktiken, die sich in der Community von JavaScript ergeben haben, konsequent umzusetzen. Dies ist unter anderem auch auf Konzepte zurückzuführen, die vor allem in CoffeeScript zum Tragen gekommen sind.
  • Block Scoped automatisch, indem Variablendeklarationen außerhalb des scopes deklariert. ES6 benutzt Let.
  • Template Strings: mithilfe von String-Interpolation, bei dem es möglich ist innerhalb eines Strings auf Variablen außerhalb des Strings zu verweisen.
  • Für einen Großteil von ES6 Funktionen lässt sich sagen, dass die Konventionen, die Entwickler bereits aus CoffeeScript kennen, übertragen lassen. Allein die Syntax unterscheidet sich etwas.
