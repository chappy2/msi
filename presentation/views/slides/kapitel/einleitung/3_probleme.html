Probleme von Javascript

  • Beeinflusst von unterschiedlichen Sprachen
    • C (imperative Syntax, {} ; )
    • Scheme (funktionale Programmierung)
    • Self (objektorientierte Programmierung, Prototypen)
  • Lexikalischer Scope
  • Typerweiterung von vielen nicht verstanden

→ CoffeeScript vereinfacht Syntax und verdeckt diese Probleme

  • JavaScript ist eine Verschmelzung von verschiedenen Ideen aus unterschiedlichen Programmiersprachen, wie…
  • Durch die verschiedenen Einflüsse hat JS viele Probleme. Das wären zum einen das Fehlen von Klassen (zumindest bis zur 5. ECMAScript Version) und die wortreiche Syntax.
  • Lexical, ein Problem das in JavaScript häufig auftritt mit der Bekanntheit von Variablen, kann zu dem ein oder anderen unerwarteten Fehler führen.
  • Die Liste von Probleme ist sehr lang. Aus diesen Problemen heraus kann es nur der Zweck einer solchen Sprache wie CoffeeScript sein, JavaScript-Probleme zu lösen. CoffeeScript verdeckt viele der genannten Probleme, mit eigener Syntax.
